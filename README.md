# Hunt-The-Duck
This is an action/advanture game about Abdo who loves the wilderness. Going every friday to hunt the ducks FOR SOLVING A HUGE CASES

To play the game you can download it ZIP and just (go live) with it using VScode or something like it (:

+ --------------------------RULES----------------------------------
+ You only have 30 seconds. Try hard to get your best SCORE!!!
+ Click on the ducks and knock them to get +10 SCORES
+ The Bad luck Ducks, its gonna be darker. -10 SCORES
+ --------------------------RULES----------------------------------

ENJOY PLAYING !!!
